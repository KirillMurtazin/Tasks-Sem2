# Проект геометрических классов

## Описание
Проект представляет собой библиотеку для работы с геометрическими объектами на плоскости. Реализованы классы для работы с точками и отрезками. Библиотека предоставляет удобный интерфейс для выполнения геометрических операций и может быть использована в других проектах.

## Структура проекта
- `Point.h/cpp` - класс для работы с точками на плоскости
  - Поддерживает создание точек с заданными координатами
  - Реализует операции сравнения точек
  - Предоставляет доступ к координатам
- `Segment.h/cpp` - класс для работы с отрезками
  - Создание отрезка по двум точкам
  - Вычисление координат точек на отрезке
  - Операции сдвига отрезка
  - Ввод/вывод отрезка
- `geometry_defines.h` - определения для экспорта/импорта библиотеки
- `tests.cpp` - модульные тесты с использованием Google Test Framework
- `main.cpp` - пример использования библиотеки
- `CMakeLists.txt` - конфигурация сборки проекта

## Сборка проекта
Для сборки проекта используется CMake. Выполните следующие команды:

```powershell
mkdir build
cd build
cmake ..
cmake --build .
```

После сборки вы получите:
- `geometry_lib.lib` - статическая библиотека с классами
- `test_lib.dll` и `test_lib.lib` - динамическая библиотека с тестами
- `main_app.exe` - исполняемый файл с примером использования

## Библиотеки
Проект состоит из двух библиотек:
1. `geometry_lib` - статическая библиотека с классами Point и Segment
   - Может быть использована в других проектах
   - Предоставляет полный API для работы с геометрическими объектами
2. `test_lib` - динамическая библиотека с модульными тестами
   - Содержит тесты для проверки корректности работы классов
   - Использует Google Test Framework

## Использование
Для использования библиотеки в своем проекте:
1. Подключите заголовочные файлы `Point.h` и `Segment.h`
2. Слинкуйте с библиотекой `geometry_lib`

Пример использования:
```cpp
#include "Point.h"
#include "Segment.h"

int main() {
    // Создание точек
    Point p1(0, 0);
    Point p2(2, 2);
    
    // Создание отрезка
    Segment seg(p1, p2);
    
    // Вычисление Y для заданного X
    double y = seg.calculateY(1.0); // y = 1.0
    
    // Сдвиг отрезка
    Segment shifted = seg << 1.0;
    
    return 0;
}
```

## Требования
- C++17
- CMake 3.10 или выше
- Компилятор с поддержкой C++17
- Google Test Framework (для тестов)

## Тестирование
Модульные тесты реализованы с использованием Google Test Framework. Тесты проверяют:
- Корректность работы конструкторов
- Правильность вычисления координат
- Работу операторов сравнения
- Обработку исключений
- Ввод/вывод объектов

Для запуска тестов используйте скомпилированный исполняемый файл `test_lib`.
